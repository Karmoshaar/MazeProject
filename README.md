مشروع توليد وحل المتاهات (Maze Generator & Solver)
هذا المشروع عبارة عن نظام متكامل بلغة Python لتوليد متاهات عشوائية وحلها باستخدام خوارزميات البحث المختلفة، مع نظام متطور لتحليل الأداء والمقارنة بين الخوارزميات من حيث السرعة واستهلاك الذاكرة.
المميزات الأساسية
• توليد المتاهات: استخدام خوارزمية Recursive Backtracking لإنشاء متاهات "مثالية" (لا تحتوي على حلقات وتضمن وجود مسار واحد).
• خوارزميات الحل: يدعم أربعة أنواع من البحث: (DFS, BFS, Dijkstra, A*) [21، 22، 23].
• تحليل الأداء: قياس الوقت بالملي ثانية، واستهلاك الذاكرة بالكيلوبايت، وعدد الخطوات في المسار [4، 5].
• التصور البياني: رسم المتاهة والمسارات بشكل مرئي باستخدام مكتبة Matplotlib [16، 18].
هيكلية المشروع (Project Structure)
ينقسم المشروع إلى وحدات برمجية منظمة:
1. maze/cell.py: يمثل الخلية الأساسية، ويخزن حالة الجدران الأربعة ومواقع الجيران.
2. maze/maze.py: يدير الشبكة ثنائية الأبعاد والربط المنطقي بين الخلايا [8، 10].
3. maze/generator.py: المسؤول عن تحويل الشبكة إلى متاهة باستخدام المكدس (Stack) [12، 13].
4. maze/solver.py: يحتوي على منطق خوارزميات البحث وإعادة بناء المسار [20، 21، 22].
5. analysis/metrics.py: أداة قياس الكفاءة باستخدام tracemalloc و time.
6. visualization/draw.py: موديول الرسم الذي يحول البيانات البرمجية إلى واجهة رسومية.
آلية العمل (Workflow)
يعمل البرنامج وفق التسلسل التالي:
1. إنشاء الشبكة: تهيئة كائن Maze بالأبعاد المطلوبة.
2. التوليد: حفر الممرات بشكل عشوائي.
3. الربط: استدعاء link_neighbors لتعريف المسارات المتاحة للخوارزميات.
4. المقارنة: تشغيل الخوارزميات الأربع وقياس كفاءتها.
5. العرض: طباعة جدول المقارنة في القنصل ورسم المتاهة مع أقصر مسار ناتج عن خوارزمية A* [2، 3].
الخوارزميات المدعومة
• DFS (Depth-First Search): يستكشف أعمق مسار ممكن قبل التراجع.
• BFS (Breadth-First Search): يستكشف الجيران طبقة بطبقة لضمان أقصر مسار [21، 22].
• Dijkstra: يبحث عن المسار الأقل تكلفة.
•  (A-Star):* يستخدم دالة تقديرية (Heuristic) لتوجيه البحث بذكاء نحو الهدف.
